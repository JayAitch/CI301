ChangeableActiveQueryList

UMLClass
Workaholic :: Model1 :: ChangeableActiveQueryList

Description

Allows target collection to be modfied at runtime, responds to attribute changes to remove and reform card list.

Generalized Elements

  • ActiveQueryListElement

Specialized Elements

Operations

Visibility Name Description
public initQueryListElement()

Obscured query list init to prevent query triggering before target is selected. Query is triggered through attribute changes. When target is changed all listeners and card elements are remove and a new card collection is formed.
public get observedAttributes()

Allow element to respond to collection-target attribute changes
public attributeChangeCallback(name, oldValue, newValue)

When modifying collection-target trigger list dump and load of the new query
public removeAllCards()

Remove any cards currently attached to this list
