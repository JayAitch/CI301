NotificationCard

UMLClass
Workaholic :: Model1 :: NotificationCard

Description

Card to display notification information to user. Does this through mirroring attribute changes to visible DOM properties.

Generalized Elements

  • HTMLElement

Specialized Elements

Operations

Visibility Name Description
public connectionCallback()

Forms basic HTML and created references to DOM elements needed to display data.
public get observedAttributes()

Use custom element API to observe attribute changes. Makes attribute changes to is-read and message fire the callback.
public attributeChangedCallback(name, oldValue, newValue)

Apply DOM modifications to display to the user. Message property display information about the notification and is-read to change styling .
public _clickHandler()

General click handler to allow generic notifications to be marked as read
public markAsRead()

applies document changes to notification object to display as read.
