Operations

Visibility Name Description
public submitForm()

Own Implementation of submision method. Uses data entered in the form to modify document data.
public populateDocumentFromForm()

Populates the new/changed object with any values entered in the form. Understands how to reprisent different input types as database objects.
public mapDataFromValueSelectMap()

creates a key:value map from a SelectAndValueField any repeated keys will override previouse ones.
public mapDataFromMultiSelect()

Map data from a multiselect field. Allows the database to represent select data in a more minmal way.
public createNewForm(type)

triggers creation of different form types. Allows objects variables to be defined externally. Forms a document from default values of the type and triggers form creation, populates form field values with type defaults.
public createFormFromExisting(type, document)

triggers form creation of specfic type uses the document pased in to populate the form field values.
public createNewTeamForm(object)

understands how to translate team fields into form elements. Creates relivant form inputs and assigns values from object to form field values.
public createNewTaskForm(object)

understands how to translate task fields into form elements. Creates relivant form inputs and assigns values from object to form field values.
public clearFormDataFields()

clear any of the fields from the form data fieldset.
