ActiveQueryListElement

UMLClass
Workaholic :: Abstract Query Lists :: ActiveQueryListElement

Description

Appends listeners to query reference to allow cards in a list to be added, modified and removed at runtime. Controls listener activity and removing when element is removed from the DOM
